## vypher version

Print the version of vypher

### Synopsis

Prints the currently installed version of the `vypher` binary.

### Usage

```
vypher version
```

### Output

```
vypher v1.2.0
```

When built from source without a version tag (e.g. `go build .`), the version will display as `dev`:

```
vypher dev
```
