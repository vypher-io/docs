# vypher docs

Open the Vypher documentation site in your default browser.

## Usage

```
vypher docs
```

## Description

`vypher docs` launches [docs.vypher.io](https://docs.vypher.io) in your system's default browser. It works on macOS, Linux, and Windows without any arguments.

## Examples

```bash
vypher docs
```

If a browser cannot be opened (e.g. in a headless environment), Vypher prints the URL to stderr so you can open it manually:

```
Could not open browser: ...
Visit: https://docs.vypher.io
```
